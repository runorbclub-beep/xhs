# 快捷启动指南

## 🚀 在任何地方快速启动

现在你可以在任何目录下直接使用快捷命令了！

## 📝 快捷命令列表

### 方式1：使用快捷脚本（推荐）

**在项目目录中：**
```bash
cd /Users/xiaoningli/xiaohongshu_automation

# 启动交互式浏览会话 ⭐
./xhs.sh --start-session

# 搜索账号
./xhs.sh --search "牙医" --auto-add

# 生成报告
./xhs.sh --report

# 查看监控列表
./xhs.sh --list
```

**在任何目录中：**
```bash
# 直接使用 xhs 命令（已创建符号链接）
xhs --start-session
xhs --report
xhs --list
```

### 方式2：切换到项目目录后使用

```bash
# 切换目录
cd /Users/xiaoningli/xiaohongshu_automation

# 使用原命令
python3 smart_social_assistant.py --start-session
```

## 🔧 故障排除

### 问题：找不到 `xhs` 命令

**解决方案：**
```bash
# 1. 确认符号链接存在
ls -la ~/xhs

# 2. 如果不存在，重新创建
ln -sf /Users/xiaoningli/xiaohongshu_automation/xhs.sh ~/xhs
chmod +x ~/xhs

# 3. 或者直接切换到项目目录
cd /Users/xiaoningli/xiaohongshu_automation
./xhs.sh --start-session
```

### 问题：权限被拒绝

**解决方案：**
```bash
# 添加执行权限
chmod +x ~/xhs
chmod +x /Users/xiaoningli/xiaohongshu_automation/xhs.sh
```

### 问题：Chrome连接失败

**解决方案：**
```bash
# 确保Chrome以CDP模式运行
# macOS
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222 &
```

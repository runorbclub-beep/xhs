# 智能社交助手 - 使用指南

## 📖 工具说明

**智能社交助手**是一个人机协作工具，帮助您在社交媒体平台上建立关系、推广业务。

### 工作模式

**✅ 您负责（真实操作）：**
- 手动登录社交媒体平台
- 查看助手提供的账号和内容建议
- 亲自点赞和发布评论

**🤖 机器人负责（辅助工作）：**
- 搜索和筛选合适的账号
- 监控账号的新作品
- 智能起草评论建议
- 生成每日互动清单

### 核心优势

✅ **零封号风险** - 机器人不发送任何内容到平台
✅ **内容质量高** - 每条评论都经过您审核
✅ **建立真实关系** - 基于真实互动，而非机器人操作
✅ **数据追踪** - 记录所有互动历史
✅ **智能建议** - 根据内容类型生成个性化评论

## 🚀 快速开始

### 第一步：添加监控账号

找到您想要关注的目标账号（例如：美容院、牙医、瑜伽馆等），然后添加到监控列表：

**命令行方式：**
```bash
python3 smart_social_assistant.py --add-account "账号名,账号ID,分类"
```

**示例：**
```bash
python3 smart_social_assistant.py --add-account "小美美容院,user123,美容院"
python3 smart_social_assistant.py --add-account "张医生牙科,user456,牙医"
python3 smart_social_assistant.py --add-account "静心瑜伽馆,user789,瑜伽馆"
```

**交互模式：**
```bash
python3 smart_social_assistant.py --interactive
# 然后选择 1. 添加账号
```

### 第二步：添加笔记到跟踪

当您发现这些账号发布了新作品时，添加笔记URL到跟踪列表，助手会自动生成评论建议：

**命令行方式：**
```bash
python3 smart_social_assistant.py --add-note "账号名,笔记ID,笔记标题,笔记URL"
```

**示例：**
```bash
python3 smart_social_assistant.py --add-note "小美美容院,note123,美容院预约管理系统分享,https://www.xiaohongshu.com/explore/note123"
```

**交互模式：**
```bash
python3 smart_social_assistant.py --interactive
# 然后选择 2. 添加笔记
```

### 第三步：查看每日报告

每天查看助手生成的互动清单：

**命令行方式：**
```bash
python3 smart_social_assistant.py --report
```

**交互模式：**
```bash
python3 smart_social_assistant.py --interactive
# 然后选择 3. 查看报告
```

报告会保存在 `reports/每日报告_YYYYMMDD.txt`

### 第四步：进行真实互动

1. 打开小红书APP或网页
2. 根据报告中的清单逐一访问笔记
3. 参考助手提供的"建议评论"
4. **亲自点赞和发布评论**（这是您的工作！）
5. 记录已完成的互动

### 第五步：标记已互动

完成互动后，标记为已完成：

**命令行方式：**
```bash
python3 smart_social_assistant.py --mark "账号名,笔记ID,互动类型"
```

**示例：**
```bash
python3 smart_social_assistant.py --mark "小美美容院,note123,评论"
```

**交互模式：**
```bash
python3 smart_social_assistant.py --interactive
# 然后选择 4. 标记互动
```

## 📋 每日工作流程

### 推荐的每日工作流程：

**🌅 早上（10分钟）：**
1. 在小红书上查看关注的账号是否有新作品
2. 发现新作品后，使用 `--add-note` 添加到跟踪列表
3. 助手会自动生成评论建议

**🌆 下午或晚上（30分钟）：**
1. 运行 `--report` 查看今日互动清单
2. 打开小红书，逐一访问清单中的笔记
3. 参考建议评论，亲自点赞和评论
4. 使用 `--mark` 标记已完成的互动

## 💡 如何找到目标账号

### 方法一：小红书搜索

1. 打开小红书APP或网页
2. 搜索关键词：
   - 美容院、美容店、美容工作室
   - 牙医、牙科、口腔诊所
   - 瑜伽馆、健身工作室
   - 按摩店、理疗馆
3. 筛选条件：
   - 粉丝数：1,000 - 50,000（不要太像营销号）
   - 近期有更新
   - 有真实分享内容（不是纯广告）
4. 记录账号名和ID

### 方法二：评论区发现

1. 在相关热门笔记的评论区寻找
2. 找到同行或目标客户的账号
3. 记录账号信息

## 📊 数据存储位置

所有数据保存在 `data/` 目录：

```
data/
├── monitored_accounts.json  # 监控的账号和笔记
└── tracked_notes.json       # 跟踪的笔记详情
```

报告保存在 `reports/` 目录：

```
reports/
└── 每日报告_YYYYMMDD.txt     # 每日互动报告
```

## ⚠️ 重要注意事项

1. **您亲自操作**：所有点赞和评论都是您手动操作，机器人只提供建议
2. **真实互动**：评论建议仅供参考，请根据具体内容调整，保持真实性
3. **质量优先**：每天5-10个高质量互动，胜过100个机械式评论
4. **建立关系**：目标是建立真实的同行关系，而不是简单的推广
5. **长期坚持**：关系建立需要时间，保持耐心和持续互动

## 🆘 常见问题

### Q: 如何获取账号ID？
A: 在小红书网页版打开账号主页，URL中的用户名部分就是账号ID。

### Q: 如何获取笔记ID？
A: 在小红书网页版打开笔记，URL中的ID部分就是笔记ID。

### Q: 可以监控多少个账号？
A: 没有限制，但建议10-20个账号为宜，保证互动质量。

### Q: 评论建议太生硬怎么办？
A: 建议仅供参考，请根据具体内容和个人风格调整。

### Q: 会不会被平台检测为机器人？
A: 不会。所有操作都是您手动完成，机器人只在本地提供建议。
